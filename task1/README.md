# Art Gallery Website

## Project Overview
A modern, responsive art gallery website designed with HTML5 semantic elements, external CSS, and advanced features including responsive design, web fonts, and creative styling.

## Features Implemented

### ✅ 1. HTML5 Semantic Elements
- `<header>` - Contains logo and navigation
- `<nav>` - Navigation bar
- `<main>` - Main content wrapper
- `<section>` - Different content sections (Home, Gallery, Featured, About, Contact)
- `<article>` - Gallery items and feature cards
- `<footer>` - Footer with social links

### ✅ 2. Type Selectors, ID & Class
- **Type Selectors**: `body`, `h1`, `h2`, `p`, `a`, `section`, `nav`, `footer`
- **ID Selector**: `#logo` for the logo section
- **Class Selectors**: `.hero-section`, `.gallery-item`, `.feature-card`, `.btn-primary`, etc.

### ✅ 3. Navigation Bar & Pseudo Classes
Implemented pseudo-classes:
- `:hover` - Changes appearance on mouse hover
- `:active` - Changes when clicked
- `:visited` - Styles visited links
- `:focus` - Keyboard navigation focus
- `:first-child` - First navigation item
- `:last-child` - Last navigation item
- `:nth-child(even)` - Even navigation items

### ✅ 4. External CSS File
- **styles.css** - Complete external stylesheet
- Linked in HTML: `<link rel="stylesheet" href="styles.css">`

### ✅ 5. Background Color/Image, Color and Font
- **Background**: Gradient backgrounds (`linear-gradient`)
- **Colors**: Custom color scheme (#667eea, #764ba2, #2c3e50, etc.)
- **Fonts**: 
  - Playfair Display (headings)
  - Raleway (body text)
  - Montserrat (accents)

### ✅ 7. Creativity
- Animated hover effects on gallery items
- Gradient backgrounds and overlays
- Filter buttons for artwork categories
- Smooth scroll animations
- Decorative elements and badges
- Social media icons
- Interactive feature cards
- Testimonial section

### ✅ 8. Responsive Web Design (Media Queries)
Media queries for:
- **Large Desktop** (min-width: 1400px)
- **Desktop** (default)
- **Tablet** (max-width: 1024px)
- **Mobile** (max-width: 768px)
- **Small Mobile** (max-width: 480px)

Changes include:
- Responsive grid layouts
- Adjusted font sizes
- Mobile-friendly navigation
- Different background gradients per device
- Flexible images and containers

### ✅ 9. Neat Typography - Web Fonts
**Google Fonts Used**:
- **Playfair Display** - Elegant serif for headings
- **Raleway** - Clean sans-serif for body text
- **Montserrat** - Modern sans-serif for accents

Font properties:
- Multiple font weights (300, 400, 600, 700, 900)
- Letter spacing for headings
- Proper line height (1.6)
- Text shadows for depth

### ✅ 10. Neat Design
- Clean, modern layout
- Consistent spacing and alignment
- Professional color palette
- High-quality images
- Well-organized sections
- Accessibility features
- Smooth transitions and animations

## How to View
1. Open `index.html` in a web browser
2. The page is fully responsive - try resizing your browser
3. All navigation links work with smooth scrolling
4. Hover over gallery items and buttons for interactive effects
